FreePBX Integration

The examples shown on this site are for an Asterisk install that uses the default configuration files. If you are using FreePBX open_in_browser www.freepbx.org to provision Cisco phones then you will need to perform additional steps to apply the device specific configuration options.

Listed below are external guides created by other people that may be useful.

open_in_browser FreePBX Wiki has a slightly outdated guide on how to patch Asterisk 11 and also has some examples of the additional configuration needed.
